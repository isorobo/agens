</objective>

**As a** developer scoping an agent project, **I want to** have framework and SDK
fit questions routed to the GSD AI-integration skill rather than answered from
agens' own knowledge, **so that** framework judgement stays grounded in the GSD
tool that owns it and agens never drifts into reimplementing it.
